Etiqueta Div

El nombre de la etiqueta "div" tiene su origen en la palabra división, ya que esta etiqueta define zonas o divisiones dentro de una página HTML. En cualquier caso casi todos los diseñadores utilizan la palabra "capa" para referirse a una "división".

Etiqueta Div y Css

Aunque se trata de un error grave (las capas se crean mediante una propiedad de CSS llamada z-index) es preferible seguir llamando "capas" a las zonas definidas con la etiqueta "div" para poder entenderse con el resto de diseñadores.
